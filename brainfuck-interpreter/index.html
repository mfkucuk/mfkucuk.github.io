">" = moves the pointer 1 cell to the right.

"<" = moves the pointer 1 cell to the left.

"+" = increment cell value by 1.

"-" = decrement cell value by 1.

"." = prints a ascii character with the value inside the cell.

"," = takes a digit input between 1 - 9.

"[" = start of a loop (marks the cell).

"]" = end of a loop (the loops ends when the marked cell has the value 0).
